# 关于

iCtrl 是一款基于网页的远程控制应用程序，支持具有图形界面和音频功能的虚拟网络控制台 (VNC) 连接、具有终端界面的安全外壳 (SSH) 连接、基于工作负载检测的机器切换，以及通过 SFTP 进行文件管理。其网页版本专为移动设备的可访问性设计，允许用户通过浏览器管理和访问自己的会话。
